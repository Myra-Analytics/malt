# Malt Ontology
This is a public repository for issues pertaining to the [Malt Ontology](https://rdf.ag/o/malt). Available here in XML/RDF, it is also published on the main website with turtle, ntriples, json-ld and schema.org-style inline serializations. Full documentation and triple descriptions are available in French and English.

Comments, Issues and Pull Requests as welcome.
